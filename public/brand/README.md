# Pasta de Marca (Brand Assets)

Esta pasta foi criada para que você possa personalizar o logotipo do aplicativo de forma simples e direta, sem precisar alterar nenhuma linha de código!

## Como alterar o Logotipo do Aplicativo:

1. Salve a imagem do seu logotipo em um dos seguintes formatos: **PNG, SVG, JPG ou WebP**.
2. Nomeie o arquivo como:
   - `logo-custom.png` (Recomendado se for imagem estática com fundo transparente)
   - ou `logo-custom.svg` (Se for um vetor)
   - ou `logo-custom.webp`
   - ou `logo-custom.jpg` (ou `logo-custom.jpeg`)
3. Envie/substitua o arquivo nesta pasta `/public/brand/`.

O sistema irá detectar automaticamente o novo arquivo e usá-lo em todas as partes do aplicativo (como na tela de login e no cabeçalho).

## Caso nenhum arquivo seja encontrado:
O aplicativo continuará exibindo o logotipo padrão animado da Liga Positiva automaticamente (efeito catavento), garantindo que nada fique quebrado se você decidir remover o arquivo personalizado.
